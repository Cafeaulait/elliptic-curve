val log : element -> element -> int
val gcd : element -> element -> element
val extended_gcd : element -> element -> element * element * element
extended_gcd x y returns a triple ( a, b, c ) s.t. ax + by = c = gcd(x,y), provided that both x and y >= 0.
val invert : element -> element
Finds the inverse element.
val sqrt : element -> element
Computes sqrt n in the Babylonian method. n must be positive.
val mass_add : element -> ('t -> 't -> 't) -> 't -> 't -> 't -> 't
mass_add n add one zero minus_one computes the summation of n elements of one.
val mass_apply : element -> ('t -> 't -> 't) -> 't -> 't -> 't
Let * be a commutative and associative binary operator. mass_apply n f u e computes u * u * ... * u or u^n provided that f is a prefix notation of * i.e. f x y = x * y. e is the unit of * i.e. x * e = x.
val power : element -> element -> element
power x n computes x^n.
