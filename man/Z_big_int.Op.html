Module Z_big_int.Op

module Op: Field.MakeGenericOperation(Core)
val log : C.element -> C.element -> int
Finds the maximum integer x s.t. b^x <= n provided that b, n >= 0.
val gcd : C.element -> C.element -> C.element
compute gcd(x,y) assuming both x and y >= 0.
val extended_gcd : C.element -> C.element -> C.element * C.element * C.element
extended_gcd x y returns a triple ( a, b, c ) s.t. ax + by = c = gcd(x,y), provided that both x and y >= 0.
val invert : C.element -> C.element -> C.element
invert a p returns a-1 mod p provided that p is a prime number and 0 < a < p.
val sqrt : C.element -> C.element
Computes sqrt n in the Babylonian method. n must be positive.
val mass_add : C.element -> ('a -> 'a -> 'a) -> 'a -> 'a -> 'a -> 'a
mass_add n add one zero minus_one computes the summation of n elements of one.
val mass_apply : C.element -> ('a -> 'a -> 'a) -> 'a -> 'a -> 'a
Let * be a commutative and associative binary operator. mass_apply n f u e computes u * u * ... * u or u^n provided that f is a prefix notation of * i.e. f x y = x * y. e is the unit of * i.e. x * e = x.
val power : C.element -> C.element -> C.element
power x n computes x^n.
val legendre_symbol : C.element -> C.element -> int
The Legendre symbol.
val find_quadratic_nonresidue : C.element -> C.element
Find a quadratic non-residue.
val quadratic_residue : C.element -> C.element -> C.element -> C.element
quadratic_residue n a p finds a quadratic residue of a mod p. n can be an arbitrary quadratic non-residue.
